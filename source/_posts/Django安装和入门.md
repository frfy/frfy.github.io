## 安装
* pip安装直接执行：

```
pip install django
```

* [下载文件](https://www.djangoproject.com/download/2.0.7/tarball/)解压后进入到文件目录，执行：

```
python setup.py install
```

* 检查安装是否成功输入：

```
python -m django --version #显示版本号
```
